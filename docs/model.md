# AI Tutor

## Objective

The AI Tutor assists students during SQL learning.

## Capabilities

- Explain SQL concepts
- Answer student questions
- Guide problem solving
- Provide personalized hints

## Prompt Engineering

The tutor receives:

- Student question
- Current lesson
- Student progress

and generates contextual responses.
