# REST API

## Authentication

POST /login

Authenticates a student.

---

## Diagnostic

POST /diagnostic/start

Starts the diagnostic assessment.

---

POST /diagnostic/submit

Submits diagnostic answers.

---

## Lessons

GET /lesson

Returns lesson content.

---

POST /lesson/verify

Checks student answers.

---

## Tutor

POST /chat

Sends a question to the AI tutor.

Returns an AI-generated explanation.
